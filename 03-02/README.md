# 03-02. Число единиц в троичной СС
## Условие tl;dr
Дано 32-битное беззнаковое число x.
Нужно вычислить функцию F(x) — количество единиц в троичной записи числа.
**Важно:** по условию функцию надо описать рекурсивно и по соглашению вызовов cdecl.

## Решение — псевдокод
``` rust 
fn f(x: u32) -> u32 {
    // База рекурсии:
    // в записи числа 0 нет единиц
    if x == 0 {
        return 0;
    }

    // last_digit = последняя цифра в троичной записи
    let last_digit = x % 3;

    // Рекурсивно считаем ответ для x / 3
    let rest = f(x / 3);

    // Если последняя цифра равна 1, добавляем 1
    if last_digit == 1 {
        return rest + 1;
    } else {
        return rest;
    }
}

fn solve() {
    read x;
    print f(x);
}
```
