# 03-07.
## Условие — tl;dr
Даны:
- K — основание системы счисления, 2 ≤ K ≤ 10
- N — номер элемента, 1 ≤ N
- A, A ≤ 4_000_000
Строится последовательность:
```
Y0 = A mod 2011
Yi+1 = @(Yi * Yi) mod 2011
```
Где @D — это число, полученное из записи D в системе счисления по основанию K, если записать её **в обратном порядке**.
Нужно вывести YN в десятичной записи.
## Решение — псевдокод
``` rust 
const MOD = 2011

fn reverse_in_base(x: u32, k: u32) -> u32 {
    // Разворачиваем запись x в системе счисления по основанию k
    // без явного хранения цифр:
    // просто по очереди снимаем младшие цифры и строим новое число
    let mut a = x
    let mut rev = 0

    while a > 0 {
        let digit = a % k
        rev = rev * k + digit
        a = a / k
    }
    return rev
}

fn solve() {
    read k, n, a
    let mut y = a % MOD

    for i in 0..n {
        let t = y * y
        let rev = reverse_in_base(t, k)
        y = rev % MOD
    }

    print y
}
```
