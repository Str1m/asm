# 03-08. Суммы цифр
## Условие — tl;dr

Даны:
- N — число, записанное в системе счисления с основанием K
- K — основание системы счисления (2 ≤ K ≤ 10)
Нужно делать так:
1. взять число
2. посчитать сумму его цифр в системе счисления с основанием K
3. записать это новое число
4. повторять, пока не получится однозначное число
Требуется вывести сумму всех чисел, которые были записаны на доске, в десятичной записи.
## Решение — псевдокод
``` rust 
fn sum_digits(mut x: u32, k: u32) -> u32 {
    let mut s = 0;

    // Сумма цифр числа x в системе счисления по основанию k
    while x > 0 {
        s += x % k;
        x /= k;
    }

    return s;
}

fn solve() {
    read n, k;

    let mut x = n;
    let mut ans = 0;

    loop {
        // текущее число уже записано на доске
        ans += x;

        // если число однозначное в базе k, заканчиваем
        if x < k {
            break;
        }

        x = sum_digits(x, k);
    }

    print ans;
}
```
