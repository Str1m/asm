# 03-03. 
## Условие tl;dr
На вход подаётся последовательность целых чисел, оканчивающаяся нулём.
Ноль в саму последовательность не входит.
Нужно вывести все числа **в обратном порядке**.
Ограничение:
- массивы использовать нельзя
- нужно сделать это рекурсией
- каждый вызов должен использовать O(1) памяти

## Решение — псевдокод
Идея:
- читаем очередное число x
- если x == 0, заканчиваем
- иначе рекурсивно обрабатываем хвост
- после возврата печатаем x

``` rust 
fn print_reversed() {
    read x;

    // База рекурсии: встретили конец последовательности
    if x == 0 {
        return;
    }

    // Сначала обрабатываем всё, что идёт дальше
    print_reversed();

    // Потом печатаем текущее число
    print x;
    print ' ';
}

fn solve() {
    print_reversed();
}
```
