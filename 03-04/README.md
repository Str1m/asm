# 03-04. 
## Условие tl;dr
Дана последовательность целых чисел, заканчивающаяся 0 (он не входит в неё).

Нужно вывести:
1. сначала все нечётные позиции (1-й, 3-й, 5-й, …) — в исходном порядке
2. затем все чётные позиции — в обратном порядке
Массивы использовать нельзя → только рекурсия.
## Решение — псевдокод
``` rust
fn solve_rec(pos: u32) {
    read x;

    // конец последовательности
    if x == 0 {
        return;
    }

    if pos % 2 == 1 {
        // нечётная позиция → печатаем сразу
        print x;
        print ' ';
    }

    // идём дальше
    solve_rec(pos + 1);

    if pos % 2 == 0 {
        // чётная позиция → печатаем после возврата
        print x;
        print ' ';
    }
}

fn solve() {
    solve_rec(1);
}
```
